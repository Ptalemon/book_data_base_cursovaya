# book_data_base_cursovaya
Это курсовая первого курса. Курсовая представляет из себя базу данных книг библиотеки.

В данной курсовой работе были реализованы возможности классов и принципов ООП.

По структурам каждого класса:

book.cpp отвечает за учёт данных о каждой книге: 
* номер типа книги, 
* название и автор, 
* общее кол-во копий, 
* сколько копий отсутствует, 
* сколько копий присутствует.

student.cpp отвечает за учёт данных о каждом студенте: 
* их номер зачётной книжки, 
* имя, 
* дата поступления на обучение, 
* группа, 
* институт.

transaction.cpp отвечает за структуру данных о транзакциях: 
* номер транзакции, 
* номер зачётной книжки студента, который участвовал в транзакции,
* номер книги, участвовавшей в транзакции,
* дата взятия книги на пользование
* дата возврата книги
* статус транзакции
* человек, исполнявший транзакцию

menu.cpp отвечает за меню, откуда и происходит управление всей базой данных, при помощи выбора команд цифрами на клавиатуре и дальнейшего ввода.

main.cpp отвечает за запуск меню и создание базы данных в файле типа dat, после чего можно взаимодействовать и создавать базу данных.
